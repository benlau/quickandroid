Detailed Description

You can create a custom status indicator by defining the indicator component.

Property Documentation

color : color

The color that instances of StatusIndicator will have. The color property in StatusIndicator will override this property when set.


read-onlycontrol : StatusIndicator

The StatusIndicator that this style is attached to.


indicator : Component

This defines the indicator in both its on and off status.
